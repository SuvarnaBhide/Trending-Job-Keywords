Job Title: Full Stack Developer
Company: ePiphany AI

About the job
At ePiphany AI, we are revolutionizing the AI-powered search engine space with BNNGPT. Our mission is to enhance how people access and engage with information using advanced AI technologies. We are looking for a talented Full-Stack Developer to join our dynamic team and help build and maintain our cutting-edge search engine.


Job Description:

We are seeking a skilled Full-Stack Developer to enhance and maintain our AI-powered search engine, BNNGPT. This role involves working on new features, improving existing functionalities, and ensuring smooth integration with various APIs and services. You'll be part of an innovative team, contributing to dynamic content creation, real-time interaction, and the seamless integration of AI models.


Responsibilities:

Feature Development: Develop and implement new features based on the roadmap, including search history, sharing results, video support, and RAG support.
API Integration: Integrate and configure different APIs for text streaming, search, and content generation.
UI Enhancement: Enhance the generative UI and maintain the existing UI components.
Database Management: Ensure smooth setup and integration of a serverless database solution.
App Framework Maintenance: Maintain and enhance the app framework built on Next.js and TypeScript.
Styling and Component Management: Ensure proper styling and component behavior using modern styling frameworks and component libraries.
Deployment: Deploy and manage live versions of the application on cloud platforms.
Troubleshooting: Troubleshoot and fix any build errors and deployment issues.
AI Integration: Ensure that the application can use verified AI models.


Tech Stack:

App Framework: Next.js
Language: TypeScript
Text Streaming / Generative UI: Vercel AI SDK
Open-source and Private GenAI Models
Headless Component Primitives: Radix UI
Styling: Tailwind CSS
